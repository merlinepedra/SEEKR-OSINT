## Get Person which does not exsist

**Curl Request:**

```sh
curl -X GET http://localhost:8080/people/100
```

**Response:**

```json
null
```

**Status Code:** 404


## Overwrite Person

**Curl Request:**

```sh
curl -X POST http://localhost:8080/person \
-H 'Content-Type: application/json' \
-d '{"id":"1"}'
```

**Response:**

```json
{
    "message": "overwritten person"
}
```

**Status Code:** 202
